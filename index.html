KATA from CODEWARS What is an anagram? Well, two words are anagrams of each other if they both contain the same letters. Write a function that will find all the anagrams of a word from a list. You will be given two inputs a word and an array with words. You should return an array of all the anagrams or an empty array if there are none. 

                                
                            def anagrams(word, words):
                                s = []
                                for i in words:
                                    if sorted(word) == sorted(i):
                                        s.append(i)
                                return s
